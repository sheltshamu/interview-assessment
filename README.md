# interview-assessment

## Requirements
For building and running the application you need:
- JDK 17
- Maven

## Building the project
`mvn clean install`

## Running the project
`mvn spring-boot:run`

## h2-console`

```
java -jar target/drones-0.0.1-SNAPSHOT.jar
```
